# Employee-Data-Analysis-Dashboard
Project Overview
Built an interactive Employee Data Analysis Dashboard in Microsoft Excel to analyze employee salary, department performance, gender distribution, and experience levels using Pivot Tables, Pivot Charts, KPIs, and Slicers.

Tools Used

Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Dashboard
Conditional Formatting
Excel Functions

Dataset

Employee Name
Age
Gender
Department
Job Title
Experience Years
Education Level
Location
Salary

Dashboard Features

Total Employees KPI
Average Salary KPI
Average Salary by Department
Average Salary by Gender
Salary Distribution
Average Salary by Experience
Interactive Slicers for dynamic filtering

Key Insights

Compared average salaries across different departments.
Analyzed salary distribution based on employee experience.
Evaluated gender-wise average salary.
Built an interactive dashboard to enable quick business insights.
